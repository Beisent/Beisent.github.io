# API Endpoints

# Build API

# POST /build

Triggers a full site rebuild.

Request:

{
  "clean": true,
  "verbose": false
}

Response:

{
  "status": "ok",
  "pages": 12,
  "duration_ms": 340
}

# Status API

# GET /status

Returns the current build status.
