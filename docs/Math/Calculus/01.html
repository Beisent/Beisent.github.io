# 实数

实数集 R 区别于其他数集的最核心的两个特征是：稠密性 与 连续性。

# 稠密性

定义： 在实数轴上，任意两个不相等的实数之间，必然存在另一个实数。

事实上，不仅存在一个，而是存在无穷多个实数。这一性质说明实数在直线上是“处处存在”的，不存在孤立的点。

# 连续性

定义： 实数轴是“完备”的，即数轴上不存在任何“洞”。

  • 有理数的局限性： 有理数集虽然也具有稠密性（任意两个有理数之间仍有有理数），但它不具备连续性。如果我们只看有理数，数轴上会布满密密麻麻的“洞”，这些洞对应的正是无理数。

# 实数的表现：

如果在实数轴上随机进行一次“分划”，这一刀必然会落在某个具体的实数上。这个数要么属于左半部分，要么属于右半部分。这说明实数轴是紧密相连的。

# 有理数的表现：

对于有理数集，我们可以构造一种特殊的切法。例如，将所有有理数按如下规则分类：

  • A组： 所有平方小于2的正有理数（及负有理数）；
  • B组： 所有平方大于2的正有理数。

当我们试图寻找这一“刀”切在哪个有理数上时，会惊奇地发现：在有理数范围内，根本没有这个数，这就说明有理数是不完备的，他并不连续。
