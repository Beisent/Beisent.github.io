# Plugins

# Overview

BeisentDocs supports a plugin system for extending functionality.

# Creating a Plugin

Create a Python module in the plugins/ directory:

def on_build(builder):
    print("Build started!")

# Available Hooks

HookDescription
on_buildCalled when build starts
on_pageCalled for each page
on_finishCalled when build completes
