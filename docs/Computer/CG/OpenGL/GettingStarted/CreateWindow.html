# 创建窗口

这里使用与LearnOpenGL不同的方式，LearnOpenGL使用Visual Studio，而我们使用命令行+CMake

OpenGL的画面不能直接渲染在你的屏幕上，一般情况下只能渲染在操作系统提供的桌面窗口上。

那么由于有多种操作系统的存在，每个操作系统提供的接口也不一样，所以OpenGL并没有能够直接处理窗口的能力，OpenGL将这些操作抽象出去，让我们自行处理窗口。我们可以手动处理，也可以使用别人封装好的库，这里使用GLFW。

# GLFW

GLFW提供定义窗口参数以及处理用户输入的能力。

# GLAD

因为OpenGL是一个规范，具体的实现是由显卡厂商提供的，由于OpenGL驱动版本众多，它大多数函数的位置都无法在编译时确定下来，需要在运行时查询。开发者需要在运行时获取函数地址并将其保存在一个函数指针中供以后使用。但是这已经有人为我们做了，我们使用GLAD这个库来加载所有OpenGL相关的函数。

打开GLAD的在线服务，将语言(Language)设置为C/C++，在API选项中，选择3.3以上的OpenGL(gl)版本（我们的教程中将使用3.3版本，但更新的版本也能用）。之后将模式(Profile)设置为Core，并且保证选中了生成加载器(Generate a loader)选项。现在可以先（暂时）忽略扩展(Extensions)中的内容。都选择完之后，点击生成(Generate)按钮来生成库文件。

这时解压这个zip压缩包，获得glad这个文件夹，记住这个文件夹的目录假设为(C:\Users\admin\Downloads\glad)

# 创建项目结构

mkdir learn_opengl
cd learn_opengl
mkdir thrid_party
cd thrid_party
git clone https://github.com/glfw/glfw.git
cp -r C:\Users\admin\Downloads\glad .
cd ..

这时候我们需要的第三方库已经准备好了
