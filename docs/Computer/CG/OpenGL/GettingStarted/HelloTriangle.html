# 你好，三角形

在OpenGL中，任何事物都在3D空间中，但是屏幕和窗口是2D的，3D坐标转为2D坐标的处理过程是由OpenGL的图形渲染管线管理的，通常管线可以被划分为两个主要的部分：将3D坐标转换为2D坐标；将2D坐标转变化实际的颜色像素。

图形渲染管线接受一组3D坐标，然后把它们转变为你屏幕上的有色2D像素输出。

图形渲染管线可以被划分为几个阶段，每个阶段将会把前一个阶段的输出作为输入。

这些阶段被叫做着色器，程序员可以通过编写着色器程序来自定义渲染管线，从而渲染出精致的图形。

正如这张图所示，OpenGL会将顶点数据输送给管线的第一个阶段：顶点着色器，顶点着色器主要是进行各种矩阵变换，将顶点变换到你期望的位置，进行一数据传输等等，再进入第二阶段几何着色器，几何着色器可以将一个图元转化为另一个图元，例如将一个点输出为一个正方形等等。再进入第三个阶段，进行图元装配，将不同的点按顺序连接好，这个阶段是不可编程的，第四个阶段光栅化，将集具体的坐标点光栅化为像素，片段着色器，对每个像素进行具体的上色操作，最后进行测试与混合

# 顶点着色器

我们先使用一个非常简单的Vertex Shader程序，在慢慢的接触的途中你自然能领悟到其中的写法了。

#version 330 core //表示使用 opengl3.3 核心模式
layout (location = 0) in vec3 aPos;// 表示位置0是输入变量aPos

void main()
{
    // glsl会将gl_Position作为程序的输出，由于我们的输入是一个3分量的向量，我们必须把它转换为4分量的
    gl_Position = vec4(aPos.x, aPos.y, aPos.z, 1.0);
}

现在，我们暂时将顶点着色器的源代码硬编码在代码文件顶部的C风格字符串中：

const char *vertexShaderSource = "#version 330 core\n"
    "layout (location = 0) in vec3 aPos;\n"
    "void main()\n"
    "{\n"
    "   gl_Position = vec4(aPos.x, aPos.y, aPos.z, 1.0);\n"
    "}\0";

着色器程序和C/C++非常的像，也必须进行编译和链接。

unsigned int vertexShader;
vertexShader = glCreateShader(GL_VERTEX_SHADER);
glShaderSource(vertexShader, 1, &vertexShaderSource, NULL);
glCompileShader(vertexShader);

通过glCreateShader创建一个着色器程序，参数是Shader类型

通过glShaderSource指定shader源码，着色器对象作为第一个参数。第二参数指定了传递的源码字符串数量，这里只有一个。第三个参数是顶点着色器真正的源码，第四个参数我们先设置为NULL。

最后编译Shader

我们可以通过如下程序检测Shader是否编译成功，这里不做主要介绍（或许未来）

int  success;
char infoLog[512];
glGetShaderiv(vertexShader, GL_COMPILE_STATUS, &success);
if(!success)
{
    glGetShaderInfoLog(vertexShader, 512, NULL, infoLog);
    std::cout << "ERROR::SHADER::VERTEX::COMPILATION_FAILED\n" << infoLog << std::endl;
}

# 片段着色器

片段着色器所做的是计算像素最后的颜色输出，为了让事情更简单，我们的片段着色器将会一直输出橘黄色。

#version 330 core
out vec4 FragColor;// 指定输出

void main()
{
    FragColor = vec4(1.0f, 0.5f, 0.2f, 1.0f);
}

创建和编译和Vertex Shader类似

const char *fragmentShaderSource = "#version 330 core\n"
    "out vec4 FragColor;\n"
    "void main()\n"
    "{\n"
    "   FragColor = vec4(1.0f, 0.5f, 0.2f, 1.0f);\n"
    "}\n\0";
unsigned int fragmentShader;
fragmentShader = glCreateShader(GL_FRAGMENT_SHADER);
glShaderSource(fragmentShader, 1, &fragmentShaderSource, NULL);
glCompileShader(fragmentShader);

glGetShaderiv(fragmentShader, GL_COMPILE_STATUS, &success);
if(!success)
{
    glGetShaderInfoLog(fragmentShader, 512, NULL, infoLog);
    std::cout << "ERROR::SHADER::FRAGMENT::COMPILATION_FAILED\n" << infoLog << std::endl;
}

# 着色器程序

在将完成两个着色器编译后，此时还不能使用，我们需要将他们链接起来作为一个Shader程序对象后才能使用，

首先我们需要创建一个Shader程序对象

unsigned int shaderProgram;
shaderProgram = glCreateProgram();

然后我们将编译后的shader链接到这个对象上

glAttachShader(shaderProgram, vertexShader);
glAttachShader(shaderProgram, fragmentShader);
glLinkProgram(shaderProgram);

当然你可以使用同样的方式检测是否链接成功

glGetProgramiv(shaderProgram, GL_LINK_STATUS, &success);
    if (!success) {
        glGetProgramInfoLog(shaderProgram, 512, NULL, infoLog);
        std::cout << "ERROR::SHADER::PROGRAM::LINKING_FAILED\n" << infoLog << std::endl;
    }

由于我们已经创建好了Shader程序对象了，所以之前的Shader对象就不需要了

glDeleteShader(vertexShader);
glDeleteShader(fragmentShader);

# 顶点输入

在开始绘制之前，我们需要定义顶点数据。OpenGL仅当3D坐标在3个轴（x、y和z）上-1.0到1.0的范围内时才处理它，所有在这个范围内的坐标叫做标准化设备坐标（NDC），范围内的坐标最终显示在屏幕上（在这个范围以外的坐标则不会显示）

float vertices[] = {
    -0.5f, -0.5f, 0.0f,
     0.5f, -0.5f, 0.0f,
     0.0f,  0.5f, 0.0f
};

如果你想让某个像素出现在你的屏幕中，那么它必须在NDC中，你可以通过在Vertex Shader中将不在NDC中的坐标变换到NDC中。

与通常的屏幕坐标不同，y轴正方向为向上，(0, 0)坐标是这个图像的中心，而不是左上角。

我们通过VBO来管理存放顶点数据的内存，这些数据需要移动到显存中，由于将数据从CPU内存中发送到GPU现存非常的慢，所以我们希望一次性尽可能发送多的数据。而不是每一个顶点发送一次。

在OpenGL中几乎所有的对象都有一个id标识，我们可以使用glGenBuffers分配一个缓冲区作为顶点缓冲区，OpenGL有很多缓冲对象类型，，顶点缓冲对象的缓冲类型是GL_ARRAY_BUFFER。使用glBindBuffer函数把新创建的缓冲绑定到GL_ARRAY_BUFFER目标上

unsigned int VBO;
glGenBuffers(1, &VBO);
glBindBuffer(GL_ARRAY_BUFFER, VBO);

然后我们指定缓冲区的数据，以及其内存结构

glBufferData(GL_ARRAY_BUFFER, sizeof(vertices), vertices, GL_STATIC_DRAW);

# 链接顶点属性

我们将顶点数据发送给GPU了，但是还没有告诉GPU我们的顶点数据中的布局是什么样子，我们的顶点数据不只有坐标数据，可能还会有每个顶点的颜色数据，法线数据，纹理坐标等等，所以我们需要解释我们顶点数据的布局。

我们目前的顶点内存布局是上图的样子

glVertexAttribPointer(0, 3, GL_FLOAT, GL_FALSE, 3 * sizeof(float), (void*)0);
glEnableVertexAttribArray(0);

glVertexAttribPointer

  • 第一个参数是顶点属性的索引，它是连接 C++ 代码数据 与 Shader变量 之间的那座“桥梁”，还记得Vertex Shader中的layout (location = 0) in vec3 aPos;吗这里的location = 0就是对应这里的数据
  • 第二个参数指定顶点属性的大小。坐标属性是一个vec3，它由3个值组成，所以大小是3。
  • 第三个参数指定数据的类型，这里是GL_FLOAT
  • 第四个参数定义我们是否希望数据被标准化(Normalize)。如果我们设置为GL_TRUE，所有数据都会被映射到0（对于有符号型signed数据是-1）到1之间。我们把它设置为GL_FALSE。
  • 第五个参数叫做步长(Stride)，它解释该组数据的内存大小
  • 第六个参数是偏移量在后面解释，这里先设为(void*)0

我们现在应该使用glEnableVertexAttribArray，以顶点属性位置值作为参数，启用顶点属性；顶点属性默认是禁用的。即使你设置了glVertexAttribPointerOpenGL也不会使用除非你使用glEnableVertexAttribArray开启

# 顶点数组对象

顶点数组对象(Vertex Array Object, VAO)可以将所有的顶点属性存储在其中，这样的好处就是当配置顶点属性指针时只需要执行一次，之后要使用的时候就直接使用VAO就行了不需要再次配置顶点属性指针，让数据可以复用。并且让不同的顶点数据切换变得非常简单，只需要绑定不同的VAO就行了。如果你有成千上万个物体需要渲染不使用VAO，每次都去操作VBO将会产生巨大的CPU开销。

创建一个VAO和创建一个VBO很类似：

unsigned int VAO;
glGenVertexArrays(1, &VAO);

使用VAO也很简单，我们要做的只有绑定VAO，再配置对应的VBO和属性指针，最后解绑拱后面使用就好了

# 索引缓冲对象

索引缓冲对象(Index Buffer Object，IBO)，举一个例子，假设我们需要绘制一个矩形

float vertices[] = {
    // 第一个三角形
    0.5f, 0.5f, 0.0f,   // 右上角
    0.5f, -0.5f, 0.0f,  // 右下角
    -0.5f, 0.5f, 0.0f,  // 左上角
    // 第二个三角形
    0.5f, -0.5f, 0.0f,  // 右下角
    -0.5f, -0.5f, 0.0f, // 左下角
    -0.5f, 0.5f, 0.0f   // 左上角
};

会发现有几个顶点重复了，这回产生多余的开销，如果是成千上万顶点的模型，这回产生巨大的浪费。所以为了解决这个问题，我们需要使用IBO。IBO的作用就是告诉OpenGL我们应该使用什么索引顺序去遍历顶点进行绘制。

float vertices[] = {
    0.5f, 0.5f, 0.0f,   // 右上角
    0.5f, -0.5f, 0.0f,  // 右下角
    -0.5f, -0.5f, 0.0f, // 左下角
    -0.5f, 0.5f, 0.0f   // 左上角
};

unsigned int indices[] = {
    // 注意索引从0开始! 
    // 此例的索引(0,1,2,3)就是顶点数组vertices的下标，
    // 这样可以由下标代表顶点组合成矩形

    0, 1, 3, // 第一个三角形
    1, 2, 3  // 第二个三角形
};

下一步我们需要创建IBO，和VBO类似，IBO也要被VAO储存所以也要放在VAO绑定和解绑之间

unsigned int IBO;
glGenBuffers(1, &IBO);
glBindBuffer(GL_ELEMENT_ARRAY_BUFFER, IBO);
glBufferData(GL_ELEMENT_ARRAY_BUFFER, sizeof(indices), indices, GL_STATIC_DRAW);

我们这时的完整VAO,VBO,IBO部分代码

    float vertices[] = {
         0.5f,  0.5f, 0.0f, 
         0.5f, -0.5f, 0.0f, 
        -0.5f, -0.5f, 0.0f, 
        -0.5f,  0.5f, 0.0f 
    };
    unsigned int indices[] = {  
        0, 1, 3,  
        1, 2, 3   
    };
    unsigned int VBO, VAO, EBO;

    glGenVertexArrays(1, &VAO);
    glGenBuffers(1, &VBO);
    glGenBuffers(1, &EBO);

    glBindVertexArray(VAO);
    
	glBindBuffer(GL_ARRAY_BUFFER, VBO);
    glBufferData(GL_ARRAY_BUFFER, sizeof(vertices), vertices, GL_STATIC_DRAW);

    glBindBuffer(GL_ELEMENT_ARRAY_BUFFER, EBO);
    glBufferData(GL_ELEMENT_ARRAY_BUFFER, sizeof(indices), indices, GL_STATIC_DRAW);

    glVertexAttribPointer(0, 3, GL_FLOAT, GL_FALSE, 3 * sizeof(float), (void*)0);

    glEnableVertexAttribArray(0);

    glBindVertexArray(0); // 解绑VAO

# 终于我们可以绘制出我们的矩形了（虽然标题是Hello Triangle^_^）

完整代码

#include <glad/glad.h>
#include <GLFW/glfw3.h>
#include <iostream>

// 回调函数：每当窗口大小改变，OpenGL渲染窗口的大小也相应改变
void framebuffer_size_callback(GLFWwindow* window, int width, int height)
{
    glViewport(0, 0, width, height);
}

// 输入处理：按下 ESC 键关闭窗口
void processInput(GLFWwindow* window)
{
    if (glfwGetKey(window, GLFW_KEY_ESCAPE) == GLFW_PRESS) {
        glfwSetWindowShouldClose(window, true);
    }
}

// 顶点着色器源码
const char* vertexShaderSource = "#version 330 core\n"
                                 "layout (location = 0) in vec3 aPos;\n"
                                 "void main()\n"
                                 "{\n"
                                 "   gl_Position = vec4(aPos.x, aPos.y, aPos.z, 1.0);\n"
                                 "}\0";

// 片段着色器源码
const char* fragmentShaderSource = "#version 330 core\n"
                                   "out vec4 FragColor;\n"
                                   "void main()\n"
                                   "{\n"
                                   "   FragColor = vec4(1.0f, 0.5f, 0.2f, 1.0f);\n"
                                   "}\n\0";

int main()
{
    // 1. 初始化 GLFW 并配置
    glfwInit();
    glfwWindowHint(GLFW_CONTEXT_VERSION_MAJOR, 3);
    glfwWindowHint(GLFW_CONTEXT_VERSION_MINOR, 3);
    glfwWindowHint(GLFW_OPENGL_PROFILE, GLFW_OPENGL_CORE_PROFILE);

    // 2. 创建窗口对象
    GLFWwindow* window = glfwCreateWindow(800, 600, "LearnOpenGL - Hello Triangle", NULL, NULL);
    if (window == NULL) {
        std::cout << "Failed to create GLFW window" << std::endl;
        glfwTerminate();
        return -1;
    }
    glfwMakeContextCurrent(window);
    glfwSetFramebufferSizeCallback(window, framebuffer_size_callback);

    // 3. 加载 GLAD 管理 OpenGL 函数指针
    if (!gladLoadGLLoader((GLADloadproc) glfwGetProcAddress)) {
        std::cout << "Failed to initialize GLAD" << std::endl;
        return -1;
    }

    // ------------------------------------------------------------------
    // 4. 构建并编译着色器程序
    // ------------------------------------------------------------------
    int success;
    char infoLog[512];

    // 顶点着色器
    unsigned int vertexShader = glCreateShader(GL_VERTEX_SHADER);
    glShaderSource(vertexShader, 1, &vertexShaderSource, NULL);
    glCompileShader(vertexShader);
    glGetShaderiv(vertexShader, GL_COMPILE_STATUS, &success);
    if (!success) {
        glGetShaderInfoLog(vertexShader, 512, NULL, infoLog);
        std::cout << "ERROR::SHADER::VERTEX::COMPILATION_FAILED\n" << infoLog << std::endl;
    }

    // 片段着色器
    unsigned int fragmentShader = glCreateShader(GL_FRAGMENT_SHADER);
    glShaderSource(fragmentShader, 1, &fragmentShaderSource, NULL);
    glCompileShader(fragmentShader);
    glGetShaderiv(fragmentShader, GL_COMPILE_STATUS, &success);
    if (!success) {
        glGetShaderInfoLog(fragmentShader, 512, NULL, infoLog);
        std::cout << "ERROR::SHADER::FRAGMENT::COMPILATION_FAILED\n" << infoLog << std::endl;
    }

    // 链接着色器程序
    unsigned int shaderProgram = glCreateProgram();
    glAttachShader(shaderProgram, vertexShader);
    glAttachShader(shaderProgram, fragmentShader);
    glLinkProgram(shaderProgram);
    glGetProgramiv(shaderProgram, GL_LINK_STATUS, &success);
    if (!success) {
        glGetProgramInfoLog(shaderProgram, 512, NULL, infoLog);
        std::cout << "ERROR::SHADER::PROGRAM::LINKING_FAILED\n" << infoLog << std::endl;
    }
    glDeleteShader(vertexShader);
    glDeleteShader(fragmentShader);

    // ------------------------------------------------------------------
    // 5. 设置顶点数据、缓冲区并配置顶点属性
    // ------------------------------------------------------------------
    float vertices[] = {
        0.5f,
        0.5f,
        0.0f, // 右上角
        0.5f,
        -0.5f,
        0.0f, // 右下角
        -0.5f,
        -0.5f,
        0.0f, // 左下角
        -0.5f,
        0.5f,
        0.0f // 左上角
    };
    unsigned int indices[] = {
        0,
        1,
        3, // 第一个三角形
        1,
        2,
        3 // 第二个三角形
    };

    unsigned int VBO, VAO, EBO;
    glGenVertexArrays(1, &VAO);
    glGenBuffers(1, &VBO);
    glGenBuffers(1, &EBO);

    // 绑定顺序：先绑定 VAO -> 绑定并设置 VBO -> 绑定并设置 EBO -> 设置属性指针
    glBindVertexArray(VAO);

    glBindBuffer(GL_ARRAY_BUFFER, VBO);
    glBufferData(GL_ARRAY_BUFFER, sizeof(vertices), vertices, GL_STATIC_DRAW);

    glBindBuffer(GL_ELEMENT_ARRAY_BUFFER, EBO);
    glBufferData(GL_ELEMENT_ARRAY_BUFFER, sizeof(indices), indices, GL_STATIC_DRAW);

    glVertexAttribPointer(0, 3, GL_FLOAT, GL_FALSE, 3 * sizeof(float), (void*) 0);
    glEnableVertexAttribArray(0);

    glBindVertexArray(0);

    // ------------------------------------------------------------------
    // 6. 渲染循环
    // ------------------------------------------------------------------
    while (!glfwWindowShouldClose(window)) {
        // 输入处理
        processInput(window);

        // 渲染指令
        glClearColor(0.2f, 0.3f, 0.3f, 1.0f);
        glClear(GL_COLOR_BUFFER_BIT);

        // 使用着色器程序并绘制
        glUseProgram(shaderProgram);
        glBindVertexArray(VAO);
        // 使用 glDrawElements 而不是 glDrawArrays 来使用索引绘图
        glDrawElements(GL_TRIANGLES, 6, GL_UNSIGNED_INT, 0);

        // 交换缓冲并轮询 IO 事件
        glfwSwapBuffers(window);
        glfwPollEvents();
    }

    glDeleteVertexArrays(1, &VAO);
    glDeleteBuffers(1, &VBO);
    glDeleteBuffers(1, &EBO);
    glDeleteProgram(shaderProgram);

    glfwTerminate();
    return 0;
}
