# Changelog

# v1.0.0

Initial release of BeisentDocs.

# Added

  • Markdown to HTML conversion
  • Code syntax highlighting with Highlight.js
  • Math formula rendering with KaTeX
  • Responsive design with dark/light theme
  • Sidebar navigation
  • Table of contents generation
  • GFM table support
  • Task lists
  • Admonition blocks (NOTE, TIP, WARNING, IMPORTANT, CAUTION)
  • Footnote support
  • Auto-generated sample documentation
  • Live-reload development server
  • File watcher for auto-rebuild

# Technical Details

Language:     Python 3.10+
Dependencies: None (stdlib only)
Build time:   < 1s for typical sites
Output:       Static HTML/CSS/JS
