# API Reference

# MarkdownParser

The core Markdown parser class.

# Constructor

parser = MarkdownParser()

# Methods

# parse(text: str) -> str

Converts Markdown text to HTML.

html = parser.parse("# Hello World")
# <h1 id="hello-world">...</h1>

# Supported Extensions

ExtensionSyntaxDescription
Fenced code ` Code blocks with language
Math (inline)\(...\)Inline math formulas
Math (block)$$...$$Block math formulas
Tables`\... \`GFM-style tables
Task lists- [x]Checkboxes
StrikethroughtextDeleted text
HighlighttextHighlighted text
Admonitions> [!TYPE]NOTE, TIP, WARNING, etc.
Footnotes[^key]Reference footnotes

# SiteBuilder

The site generator class.

# Constructor

builder = SiteBuilder(
    docs_dir="docs",      # Markdown source directory
    dist_dir="dist",      # Output directory
    static_dir="static",  # Static assets
    templates_dir="templates"  # HTML templates
)

# Methods

# build()

Builds the entire site. Collects all .md files from docs/, converts them, and writes the output to dist/.

builder = SiteBuilder()
builder.build()

NOTE

If no .md files exist in docs/, sample documents are auto-generated.

# CLI Usage

# Build the site
python build.py

# Serve locally
python build.py serve

# Watch mode (auto-rebuild)
python build.py watch
